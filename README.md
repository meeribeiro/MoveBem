MoveBem é um aplicativo mobile desenvolvido em Flutter que auxilia usuários a manter uma rotina saudável através de exercícios de alongamento e controle de hidratação.

O MoveBem foi desenvolvido como projeto acadêmico do curso de Dispositivos Móveis, implementando uma solução completa de saúde e bem-estar com foco em:

Rotinas de exercícios personalizadas</br>
Acompanhamento de hidratação diária</br>
Histórico detalhado de atividades</br>
Interface intuitiva e responsiva</br>

Funcionalidades:</br>
Autenticação</br>
Login seguro com criptografia bcrypt</br>
Cadastro de novos usuários</br>
Recuperação de senha</br>
Sessão persistente (auto-login)</br>

Exercícios:</br>
Visualização de rotinas de alongamento</br>
Cronômetro regressivo para cada exercício</br>
Intervalo automático de descanso (20s)</br>
Pausar/retomar treinos</br>
Registro automático de tempo total</br>


Controle de Água:</br>
Registro de consumo de água (ml)</br>
Acompanhamento diário</br>
Histórico detalhado com horários</br>

Histórico:</br>
Calendário interativo</br>
Visualização de tempo total de exercícios por dia</br>
Histórico completo de hidratação</br>
Estatísticas diárias</br>

Perfil:</br>
Edição de nome</br>
Alteração de senha</br>
Exclusão de conta</br>
Logout</br>

Tecnologias Utilizadas:</br>
Frontend (Mobile)</br>
Flutter - Framework multiplataforma</br>
Dart - Linguagem de programação</br>
HTTP - Requisições REST API</br>
SharedPreferences - Armazenamento local</br>
TableCalendar - Calendário interativo</br>
Backend (API)</br>
PHP - Linguagem server-side</br>
PostgreSQL - Banco de dados relacional</br>
PDO - Acesso seguro ao banco</br>
Bcrypt - Criptografia de senhas</br>
